## 控制 Controls

| Key      | Operation                                              |
| -------- | ------------------------------------------------------ |
| WASD     | 移动镜头 Move Camera                                   |
| ↑↓←→     | 旋转物体 Rotate Model                                  |
| 鼠标移动 | 旋转镜头 Rotate Camera                                 |
| F1~F3    | 点亮三个不同方向的平行光照 Flip the directional lights |



## 文件 Files

`src`：源代码	Source Code

`lib`：第三方库	3rd-party

`data`：模型、纹理、shader、背景图	Model/Texture/Shader/Background HDR

`assimp-vc142-mt.dll`：assimp动态库
